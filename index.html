The Tool

What it is

A simple, internal web tool for the Al Arabiya graphics team. A designer uploads an image, chooses a preset, and the system returns an AI-transformed version that's ready for air — then downloads it. It runs on the channel's own AI subscriptions, on the team's own network.

1

Upload → Preset → Result. That's the whole flow.

2

No technical skill needed — it's built for designers, not engineers.

3

Everything self-serve: the team controls its own styles.

Styles

Presets — your saved looks

Every style the tool can apply is saved as a "preset." A designer just picks one and the system applies that exact treatment — no prompt-writing, no guesswork.

A name, description, and the AI model to use

Whether it's an image or video style

An optional locked style reference so every frame in a set matches

A new preset can be added in minutes from a settings page — no developer, no waiting for a new release.

The presets screen — create, edit, and manage the available styles.
